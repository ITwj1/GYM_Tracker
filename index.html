Programming Principles

Evidence-based
Frequency
Each muscle trained 3× per week. Meta-analyses show this boosts size and strength ~50% vs. once-weekly training at equated volume.
Volume
6–8 hard sets per major muscle per session. Weekly total 12–18 sets — middle of the proven effective range for trained lifters.
Rep Range
Compounds 6–10, isolations 10–15, stability work 12–15. Every set taken 1–3 reps shy of failure.
Progression
Add one rep per set each session until the top of the range, then add weight and start at the bottom.
Correction
Face pulls, wall slides, side planks, and bird dogs appear every session. Warm-ups and primers, not extras.
Rest
2–3 min between compounds, 60–90 sec between isolations.

How to Schedule Your Week

Flexible
Rule: complete A, B, and C once each per week, in any order, with at least one day between sessions where possible.
Option 1
Mon A · Wed B · Fri C
Option 2
Tue A · Thu B · Sat C
Option 3
Weekend-heavy: Sat A · Sun B · Wed C
Back-to-back days are fine if needed. Just avoid doing the same session twice in a row.
